# BuscadorNewsAPI
### Para este proyecto se uso
* Node `18.12.1`
* NPM `8.19.2`
* Create React App

### Instalar aplicacion
1. Clonar repositorio
```zsh
git clone https://github.com/signoli/buscador-info.git
```

2. Moverse a la carpeta del proyecto
```zsh
cd buscador-info
```

3. Instalar dependencias
```zsh
npm install
```

4. Correr aplicacion
```zsh
npm start
```
